# Cesium + Next.js 15

Minimal Next.js 15 example that renders a CesiumJS globe. Only a valid Cesium Ion access token is required.
---

## Quickstart

1. Install dependencies

```bash
npm install
```

2. Create `.env.local` with your Cesium Ion token

```env
NEXT_PUBLIC_CESIUM_ION_TOKEN="YOUR_CESIUM_ION_TOKEN"
```

Get a token at https://cesium.com/ â†’ Access Tokens.

3. Run dev server

```bash
npm run dev
```
Open http://localhost:3000/.
